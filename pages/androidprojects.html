LethsMarket

Realmente ambicioso, e tomará um tempo. Minha mãe queria um aplicativo pra ajudar ela a tomar conta do mercado pequeno dela em casa, então eu decidi fazer um pra ela. Eu digo ambicioso porque eu quero ele completo, afinal é pra mãe né?. Pelas telas já imagina que eu quero fazer estatísticas, historico, controle de fiados etc. Esse é o projeto que estou trabalho atualmente e está andando muito bem, logo eu termino.

Esta Activity até o momento é o menu, ela me parece bonito, mas alguma das cores me incomoda, o menu está todo funcional e redirecionando pro Fragment correto. Estou utilizando o formato MVVM(Model View View-Model) para organização de estrutura projeto.

O login é meramente ilustrativo haha, eu fiz o login com a API da Google justamente pra não precisar inserir dados. Foi a minha primeira vez mexendo com API da google, Firebase Authentication e Android Studio, como sempre, eu gosto de me dar desafios e o conclui.

O Fragment de listagem de produtos já está concluida, falta retoques visuais, mas os dados estão sendo tradizos do Firestore Database.

A Activity responsável por cadastrar produto também já está concluida, produtos sendo inseridos corretamente no Firestore Database. Inserção e Pesquisa estão concluidos neste projeto, como ele é grande eu não o considarerei um CRUD apenas. Ainda em desenvolvimento.
